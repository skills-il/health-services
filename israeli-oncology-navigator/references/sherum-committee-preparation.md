# Preparing for the Special Services Allowance (שר"מ) Committee

This reference covers the ADL (Activities of Daily Living) framework that שר"מ committees use to determine the rate after the initial 50% paper track expires. The skill's main file (Step 3b-ii) gives a high-level summary; this file is the full prep guide.

## The 6 ADL categories

The committee scores each category 0 (independent), 1 (partial help needed), or 2 (full help needed):

| ADL Category | Hebrew | What is assessed |
|--------------|--------|-----------------|
| Mobility | ניידות | Moving within the home, transfers, stairs |
| Dressing | הלבשה | Putting on/removing clothes, shoes, buttons |
| Bathing | רחצה | Washing body, getting in/out of shower, drying |
| Eating | אכילה | Preparing immediate food, feeding self, drinking |
| Toileting | שירותים | Using toilet, cleaning self, managing incontinence |
| Continence | שליטה על סוגרים | Bladder and bowel control |

## Rate thresholds (high-level)

| Functional state | Rate |
|------------------|------|
| Help with most ADLs most hours | 50% (paper-track entry; baseline) |
| Help with ALL ADLs most hours OR constant supervision | 112% |
| Help with ALL ADLs all hours | 188% |
| Home hospice / terminal | 235% |

## Worst-day evaluation principle

The committee should evaluate based on **worst-case days**, not best days or averages. For an active-treatment patient, this means:

- **Chemotherapy patient:** days 1-7 of a cycle (peak nausea, neutropenia, fatigue), not days 14-21
- **Radiation patient mid-course:** weeks 4-6 of a 6-7 week course (peak skin reactions, fatigue, mucositis), not weeks 1-2
- **Post-surgical patient:** the first 2-4 weeks post-op when mobility is most restricted

Document the worst-day pattern explicitly in the committee package: photographs of skin reactions, drug logs showing rescue antiemetic use, blood counts showing nadir, daily diary of fatigue/cognitive episodes.

## Treatment-side-effects-as-impairments mapping

These map clinical realities to ADL categories for committee documentation:

### Chemotherapy
| Side effect | ADL category impacted |
|-------------|----------------------|
| Severe nausea / vomiting | Eating |
| Neutropenia (infection isolation) | Bathing, toileting (need supervised hygiene), constant supervision criterion |
| Peripheral neuropathy | Mobility, dressing (buttons), eating (utensils), bathing (slip risk) |
| Severe fatigue | Mobility, all transfers |
| Cognitive fog ("chemo brain") | All categories indirectly; constant supervision criterion |
| Diarrhea / incontinence | Continence, toileting |

### Radiation
| Side effect | ADL category impacted |
|-------------|----------------------|
| Head/neck mucositis | Eating (cannot chew/swallow) |
| Lymphedema | Mobility, dressing, bathing |
| Skin burns | Bathing, dressing |
| Fibrosis (neck, breast, pelvic) | Mobility, dressing |
| Profound fatigue | Mobility |

### Surgery
| Surgery type | ADL category impacted |
|--------------|----------------------|
| Major abdominal | Mobility, transfers (4-6 weeks) |
| Stoma (colostomy/ileostomy) | Toileting, continence (long-term) |
| Head/neck (laryngeal, pharyngeal) | Eating, communication |
| Mastectomy with lymph node dissection | Mobility (arm), dressing |
| Limb amputation | Mobility, transfers, dressing, bathing |
| Prostatectomy / cystectomy | Continence |

### CNS / brain tumors
| Issue | ADL category impacted |
|-------|----------------------|
| Cognitive impairment | All categories indirectly; constant supervision |
| Seizure risk | Constant supervision criterion |
| Motor deficits | Mobility, dressing, eating |
| Vision loss | Mobility, eating, bathing |

### Bone marrow transplant
| Issue | ADL category impacted |
|-------|----------------------|
| Prolonged immunosuppression | Bathing, toileting (sterile precautions) |
| GVHD (skin, gut, liver) | Skin = bathing/dressing; gut = continence/eating |
| Total body weakness | Mobility, all transfers |

## Constant supervision (השגחה מתמדת) as an independent qualifier

Even if individual ADL scores would not reach 112%, **constant supervision needs alone qualify**. Document:
- Fall risk (post-chemo neuropathy, post-surgery weakness, CNS deficits)
- Syncope / orthostatic hypotension
- Infection risk during neutropenia (cannot be left alone in public/unhygienic settings)
- Seizure risk (CNS tumors, brain metastases, post-craniotomy)
- Cognitive impairment that risks self-harm (wandering, medication errors, leaving the stove on)

Constant supervision is a binary qualifier: **either the patient needs it most/all hours, or they do not**. Frame the evidence around continuity of need, not severity.

## "Summoned vs self-requested" matters

If the patient was **summoned by NII** for a committee review (rather than the patient self-requesting), the committee opening statement should note this. The committee did not initiate the reassessment because of an obvious change; the patient is appearing only because NII asked. This subtly shifts the burden of demonstrating new functional change.

## Who builds the package

In order of cost-and-experience:

1. **Hospital oncology social worker (עו"ס אונקולוגי)** — first call. Free. Knows the local committee culture and what evidence wins.
2. **ICA Rights Center (מרכז מיצוי זכויות)** — embedded in 10+ hospitals, helpdesk 03-5721678, shikum@cancer.org.il. Free. Specialized in NII rights including שר"מ.
3. **Disease-specific NGO with in-house attorney**:
   - One in Nine for breast cancer: 03-602-1717 ext 2
   - Other diagnosis-specific groups via the oncology social worker
4. **Patients' Rights Association** — patients-rights.org. General representation.
5. **Private labor-law / disability-law attorney** — only when above options are exhausted, since fees apply.

## Documents to assemble for the committee

- Treating oncologist's detailed functional letter (NOT a generic medical summary; explicitly maps treatment side effects to ADL impact)
- Occupational therapist evaluation (if the hospital provides one)
- Daily diary (2-4 weeks) showing worst-case pattern
- Drug log: rescue antiemetics, opioids for pain, steroids for fatigue
- Blood counts showing chemo nadir if relevant
- Photos of skin reactions, lymphedema, surgical sites if relevant
- Family caregiver statement describing supervision needs
- Prior NII determinations (if appealing) and the original committee minutes
